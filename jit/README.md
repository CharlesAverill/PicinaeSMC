# jit

This repository builds an example of [self-modifying code](https://en.wikipedia.org/wiki/Self-modifying_code) for 32-bit RISC-V.
It is a [JIT](https://en.wikipedia.org/wiki/Just-in-time_compilation) compiler for a Turing-complete [Minsky machine](https://en.wikipedia.org/wiki/Counter_machine).
